# New Hacker News

## Introduction

The webapp is an attempt to clone the website news.ycombinator.com .
The data is fetched using the public APIs published.

## Tech stack

The app has been build using vite+react and tailwind css for styling.

## Steps to run

```
npm install
npm run dev
```
